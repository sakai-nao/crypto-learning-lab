# crypto-learning-lab

情報Ⅰ向け 暗号方式体験Webアプリ  
（共通鍵暗号方式／公開鍵暗号方式）

---

## 📌 概要

このWebアプリは、高校「情報Ⅰ」の授業で  
**共通鍵暗号方式**と**公開鍵暗号方式**の仕組みを体験的に理解するための教材です。

ブラウザのみで動作し、サーバー不要・オフライン利用可能です。

---

## 🔐 実装内容

### 1️⃣ 共通鍵暗号方式（Symmetric Key）

- 同じ鍵（パスフレーズ）で暗号化・復号  
- 鍵が一致しないと復号できない体験  
- 鍵配送問題への導入が可能  

### 2️⃣ 公開鍵暗号方式（Asymmetric Key）

- 受信者が鍵ペア（公開鍵・秘密鍵）を生成  
- 公開鍵で暗号化  
- 秘密鍵でのみ復号可能  
- 「公開しても安全」な鍵の概念を体験  

※RSAは学習用の小さい数値で実装（実用暗号ではありません）

## 🎓 授業での活用例

### 共通鍵暗号

- 2人組で鍵を決める  
- 暗号文をコピーして送信  
- 鍵を1文字変えて復号失敗を体験  

### 公開鍵暗号

- 受信者が公開鍵を提示  
- 送信者が暗号化  
- 秘密鍵でのみ復号できることを確認
